# 📊 Análise de Vendas com Python e Pandas

Análise exploratória de um relatório de vendas de múltiplas lojas 
espalhadas pelo Brasil, cobrindo todo o pipeline: do carregamento 
e limpeza dos dados até agrupamentos, rankings e visualizações.

## 🗂️ Sobre o Dataset
- **Fonte:** Dataset fictício para fins de estudo
- **Tamanho:** +100.000 registros | 8 colunas
- **Conteúdo:** Pedidos de vendas com informações de loja, produto, 
quantidade, preço, data e gerente responsável

## 🔧 Etapas da Análise
- Carregamento de arquivos CSV e Excel com Pandas
- Limpeza de dados: nulos, duplicatas, tipos e padronização de texto
- Criação de colunas calculadas: faturamento, região e forma de venda
- Agrupamentos e rankings por loja e produto
- Merge de tabelas para análise de metas por gerente
- Visualização de faturamento por período

## 🛠️ Tecnologias Utilizadas
- Python 3
- Pandas
- NumPy
- Jupyter Notebook
